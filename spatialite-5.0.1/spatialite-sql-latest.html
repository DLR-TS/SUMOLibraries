  • '1234'
  • '+5678'
  • '-4321'
  • '      +12345678'

The function returns 1 if TRUE and 0 if FALSE; -1 is returned when the argument is not a Text string.

  • '1234.56'
  • '+5678.09'
  • '-4321.67'
  • '-4321.67e12'
  • '6789.123E+6'
  • '      +1234.56e-6'
