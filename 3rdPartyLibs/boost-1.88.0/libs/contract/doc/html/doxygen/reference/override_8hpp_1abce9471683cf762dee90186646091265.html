Macro BOOST_CONTRACT_NAMED_OVERRIDE

BOOST_CONTRACT_NAMED_OVERRIDE — Declare an override type trait with an arbitrary name.

Synopsis

// In header: <boost/contract/override.hpp>

BOOST_CONTRACT_NAMED_OVERRIDE(type_name, func_name)

Description

Declare the override type trait named type_name to pass as an explicit template parameter to boost::contract::public_function for public function overrides.

See Also:  Named Overrides

Parameters:

func_name

Function name of the public function override. This macro is called just once even if the function name is overloaded (the same override type trait is used for all overloaded functions with the same name, see Function Overloads). (This is not a variadic macro parameter but it should never contain commas because it is an identifier.)

type_name

Name of the override type trait this macro will declare. (This is not a variadic macro parameter but it should never contain commas because it is an identifier.)
