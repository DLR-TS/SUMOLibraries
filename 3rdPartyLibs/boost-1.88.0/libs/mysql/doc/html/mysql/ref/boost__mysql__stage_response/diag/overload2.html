stage_response::diag (2 of 2 overloads)

Retrieves the contained diagnostics (rvalue reference accessor).

Synopsis 
diagnostics
diag() && noexcept;
Description

If *this contains an error, retrieves the associated diagnostic information by moving it. Otherwise (if this->has_statement() || this->has_results()), returns an empty (default-constructed) error.

Exception safety

No-throw guarantee.
