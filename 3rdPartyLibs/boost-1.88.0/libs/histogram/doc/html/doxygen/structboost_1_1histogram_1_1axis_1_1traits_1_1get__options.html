Description

Doxygen does not render this well. This is a meta-function (template alias), it accepts an axis type and returns the boost::histogram::axis::option::bitset.

If Axis::options() is valid and constexpr, get_options is the corresponding option type. Otherwise, it isboost::histogram::axis::option::growth_t, if the axis has a methodupdate, elseboost::histogram::axis::option::none_t.
