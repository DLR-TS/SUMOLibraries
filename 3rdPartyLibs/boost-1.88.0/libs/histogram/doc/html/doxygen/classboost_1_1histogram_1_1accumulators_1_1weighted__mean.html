Description

Uses West's incremental algorithm to improve numerical stability of mean and variance computation.

    Return effective counts.

    This corresponds to the equivalent number of unweighted samples that would have the same variance as this sample. count() should be used to check whethervalue() andvariance() are defined, see documentation ofvalue() andvariance().count() can be used to compute the variance of the mean by dividingvariance() bycount().
