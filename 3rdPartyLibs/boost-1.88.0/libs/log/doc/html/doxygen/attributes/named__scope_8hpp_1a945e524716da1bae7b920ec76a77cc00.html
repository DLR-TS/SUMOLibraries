Macro BOOST_LOG_FUNC

BOOST_LOG_FUNC

Synopsis

// In header: <boost/log/attributes/named_scope.hpp>

BOOST_LOG_FUNC()

Description

Macro for function scope markup. The scope name is constructed with help of compiler and contains the current function name. It may be shorter than what BOOST_LOG_FUNCTION macro produces. The scope name is pushed to the end of the current thread scope list.

Not all compilers have support for this macro. The exact form of the scope name may vary from one compiler to another.
