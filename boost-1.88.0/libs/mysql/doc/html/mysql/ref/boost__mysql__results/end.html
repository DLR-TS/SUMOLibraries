results::end

Returns an iterator pointing to one-past-the-last resultset that this object contains.

Synopsis 
results::iterator
end() const noexcept;
Preconditions

this->has_value() == true

Exception safety

No-throw guarantee.

Object lifetimes

The returned iterator and any reference obtained from it are valid as long as *this is alive. Move operations invalidate iterators.

Complexity

Constant.
