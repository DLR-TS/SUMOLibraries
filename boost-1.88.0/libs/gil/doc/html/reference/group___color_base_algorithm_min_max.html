Detailed Description

Equivalents to std::min_element and std::max_element for homogeneous color bases.

Example:

rgb8_pixel_t pixel(10,20,30);
assert(pixel[2] == 30);
static_max(pixel) = static_min(pixel);
assert(pixel[2] == 10);
