Macro BOOST_TTI_HAS_CLASS_GEN

BOOST_TTI_HAS_CLASS_GEN — Generates the macro metafunction name for BOOST_TTI_HAS_CLASS.

Synopsis

// In header: <boost/tti/gen/has_class_gen.hpp>

BOOST_TTI_HAS_CLASS_GEN(name)

Description

name = the name of the class/struct.

returns = the generated macro metafunction name.
