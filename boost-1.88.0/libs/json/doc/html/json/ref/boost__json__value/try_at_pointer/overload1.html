value::try_at_pointer (1 of 2 overloads)

Access an element via JSON Pointer.

Synopsis 
boost::system::result< value const & >
try_at_pointer(
    string_view ptr) const noexcept;
Description

This function is used to access a (potentially nested) element of the value using a JSON Pointer string.

Complexity

Linear in the sizes of ptr and underlying array, object, or string.

Exception Safety

No-throw guarantee.

Parameters

Name

Description

ptr

JSON Pointer string.

Return Value

boost::system::result<value&> containing either a reference to the element identified by ptr or a corresponding error_code.

See Also

RFC 6901 - JavaScript Object Notation (JSON) Pointer.
